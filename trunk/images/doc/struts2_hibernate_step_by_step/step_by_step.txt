1.创建一个 struts2_demo的 web project
2.拷贝rapid-framework至项目根目录
3.设置项目的编码为utf-8
4.设置源码路径
5.将generator/lib/jars加入classpath
6.安装插件 hibernate及struts2
7.修改generator.properties的jdbc数据库连接信息,其它属性暂不修改
7.1 在mysql数据库中创建表user_info
CREATE TABLE `user_info` (
  `user_id` bigint(20) NOT NULL AUTO_INCREMENT,
  `username` varchar(50) DEFAULT NULL,
  `password` varchar(50) DEFAULT NULL,
  `birth_date` date DEFAULT NULL,
  `sex` tinyint(4) DEFAULT NULL,
  `age` int(11) DEFAULT NULL,
  PRIMARY KEY (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8;

8.运行代码生成器 GeneratorMain.java
9.拷贝生成的代码至项目根目录,并将 java_src,java_test加入源代码路径
10.修改src/resources/jdbc.properties的数据库连接信息
11.部署应用struts2_demo至tomcat并启动tomcat服务器
12.访问URL: http://localhost:8080/struts2_demo/pages/UserInfo/list.do
