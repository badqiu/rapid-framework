一.代码生成器模板可以引用的相关变量
	1. g.generateByTable("table_name") 方法可以引用的变量
		table : cn.org.rapid_framework.generator.provider.db.model.Table
		env : 系统环境变量
		system : java System.getProperties() 
		generator.properties 文件中的所有属性,直接引用,没有前缀
	
	2. g.generateByClass("class") 方法可以引用的变量
		clazz : cn.org.rapid_framework.generator.provider.java.model.JavaClass
		env : 系统环境变量
		system : java System.getProperties() 
		generator.properties 文件中的所有属性,直接引用,没有前缀


二.每个模板有gg变量可以控制自身的自定义配置,如是否生成,是否覆盖目标文件
	示例: ${gg.setIgnoreOutput(true)}, 参考: http://code.google.com/p/rapid-framework/wiki/rapid_generator_gg


三.模板目录的二进制文件如 zip,rar,doc文件将会自动拷贝至输出目录,不会破坏文件格式   (通过扩展名自动识别)

四.模板引擎为freemarker,语法参考: http://freemarker.sourceforge.net/docs/index.html

五. Freemarker模板可以自动include同级目录的: macro.include文件,如果这个文件存在的话
	 举例: 如你的模板为 java_src/com/project/UserDao.java, 将自动include: java_src/com/project/macro.include 及 macro.include

六. 自动替换generator.properties中的句号(.)为反,设置key为key+"_dir"后缀,示例: pkg=com.company => pkg_dir=com/company

七. 在线文档: http://code.google.com/p/rapid-framework/wiki/rapid_generator
